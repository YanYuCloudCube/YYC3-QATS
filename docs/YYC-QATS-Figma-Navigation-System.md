## 一、Figma组件样式精准标注模板（可直接复用，含标注规范）
### 标注核心规则
1. 标注图层独立于设计图层，命名为「标注-组件名称」，锁定不影响设计；
2. 尺寸标注用蓝色（#4299E1），间距标注用橙色（#ED8936），颜色标注用色块+色值，字体标注用文字说明（字体/字号/字重/色值）；
3. 所有标注对齐组件边缘，无重叠，标注线长度统一（≤80px），标注文字靠近标注线；
4. 复用组件仅标注一次，页面中仅标注个性化布局间距，无需重复标注组件样式。
### 核心组件标注示例（含样式参数+标注方式）

### 通用标注速查表格
|标注类型|颜色值|标注方式|适用场景|
|-|-|-|-|
|尺寸标注|#4299E1|线性标注，标注值在标注线中间|组件宽高、图标尺寸、按钮尺寸|
|间距标注|#ED8936|线性标注，标注值在标注线中间|内边距、外边距、组件间距、文字间距|
|颜色标注|对应色值|16*16px色块+右侧色值（十六进制）|背景色、文字色、边框色、图标色|
|字体标注|#8892B0|字体家族+字号+字重+色值|所有文字（标题/正文/辅助文字）|
|圆角/弧度|#718096|圆角标注+数值（如圆角8px）|卡片、按钮、弹窗、图表|
|阴影标注|#718096|阴影层级+参数（如阴影2级：0 4px 12px 0 rgba(0,0,0,0.15)）|卡片、弹窗、浮层|

## 三、Figma设计落地检查清单（设计完成后校验，确保符合规范）
### 1. 设计系统校验
- [ ] 所有颜色均从色彩体系中选取，无自定义色值
- [ ] 所有字体均遵循字号/字重规范，无自定义字体
- [ ] 所有阴影均为0-4级，无自定义阴影参数
- [ ] 所有间距均为基础间距（4/8/12/16/24/32px）倍数，无自定义间距
### 2. 组件复用校验
- [ ] 所有页面元素均从组件库复用，无重复设计组件
- [ ] 组件样式统一，无个性化修改（如需修改，同步更新组件库）
- [ ] 组件交互状态完整（默认/hover/active/禁用），无缺失
### 3. 页面布局校验
- [ ] 符合全局12栅格布局，核心内容区无横向滚动条
- [ ] 模块专属布局适配正确，核心功能区占比合理
- [ ] 响应式适配完成，1366/1920/2560分辨率均正常显示
### 4. 交互逻辑校验
- [ ] 所有异步操作均有加载态，无空白等待
- [ ] 所有高危操作均有二次确认，交易/风控操作符合逻辑
- [ ] 所有交互反馈完整（点击/悬停/选中），反馈即时（100ms内）
- [ ] 核心页面（行情/策略/交易/风控）交互动效已制作
### 5. 标注规范校验
- [ ] 所有页面/组件均完成精准标注，无模糊标注
- [ ] 标注图层独立，未影响设计图层
- [ ] 标注类型（尺寸/间距/颜色/字体）区分正确，符合标注规范
### 6. 命名规范校验
- [ ] 所有文件/页面/组件/图层命名统一（英文+数字），无中文
- [ ] 命名按模块/功能分类，便于团队协作查找
- [ ] 组件库命名与页面中组件实例命名一致
